# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe Game** built using **HTML, CSS, and JavaScript**.

## 🚀 Features

- 2 Player Game
- O and X turn-based gameplay
- Winner detection
- Winner animation using GIF
- Trophy display after winning
- Simple and clean UI

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
